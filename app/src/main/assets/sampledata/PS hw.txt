About masks: black blocks; white reveals!
Aquela questão de trabalhar com um pincel preto ou branco em um layer pra revelar o que está atrás dele tem a ver com essa propriedade das máscaras. Claro que se vocÊ tem a intenção de pintar um canto com um pincel pra revelar o layer que está por trás do layer atual, precisa add ao layer atual uma linda "mask". Ela já aparece toda branca, significando que está revelando a poha toda, usar o pincel preto para esconder o que quiser e ali fazer aparecer oque está por trás.

No pincel: clique uma vez; mova o mouse para um outro ponto; mantenha shift pressionada e então clique lá para criar uma linha reta.

Para selecionar regiões por cor: Select>Color Range
Em Select > Grow / Similar: estas são funções ligadas diretamente à ferramenta magic wand, grow selecionar áreas de uma mesma tonalidade COLADAS à região já selecionada; similar seleciona todas aquelas, perto ou longe, de uma mesma tonalidade que a dela. 

Em Select > Coloro Range, há uma nova opção que é a de detectar tons de pele. Uma graça :)

Em "Surface Blur" temos um blur mais inteligente que o gaussian blur, este é usado para identificar regiões de rosto.

Se a gente usa uma modificação nos Levels pra dar mais contraste em um rosto, mas aí acaba ficando em uma tonalidade muito alaranjada, podemos usar no novo layer que surgiu junto com os levels que aplicamos uma forma diferente de blending que não seja o "Normal". Podemos usar o "luminosity", este sim, conserva só as regiões de preto e branco que queriamos adicionar à imagem.


Às vezes a gente usa muito do patch tool e termina com um resultado bastante artificial, em um rosto por exemplo. Neste caso é interessante usar o "history brush tool" para voltar a revelar um pouco dos detalhes que apagamos (claro que se redefine a opacidade desta ferramenta). Se a gente usa esse brush com opacidade de 20%, é legal passar no local umas duas vezes para revelar (2x20%=) 40% da pele de volta em mulheres, ou até 3 vzs para ficar em 60%.

Uma coisa importante pra modificar as proporções de um rosto, como um queixo duplo: Se usa a ferramenta"forward warp tool" dentro do filtro "liquify". Após a imagem melhorada como queremos, voltamos um passo dentro do painel history e clicamos no ícone de câmera, isso vai gerar um "snapshot 1" dentro do history, beleza deixa aí. Volte a clicar na ação "liquify" dentro do history.  Agora com a ferramenta history brush tool podemos revelar detalhes da imagem original e melhorar a qualidade da mudança feita com ajuda do liquify.

Meus jovens, vou ensinar-lhes a mudar o tom ou tirar uma maquiagem colorida dos zóio. 
Use o Eyedropper Tool para selecionar uma região do rosto (pode ser mais clara ou mais escura, preste atenção nisso para o resultado e tem que ser Sample Size>3 by 3 Average). 
Selecione Brush>Mode>Color, só pintar por cima da região colorida. A pele colorida volta a ter o tom de pele selecionado (que fica mais claro ou escuro, dependendo do que vc selecionou com o eyedropper).
Nunca se esqueça que é bom ir alterando a opacidade.
E o Burn tool é bom pra usar nos cílios e deixar mais escuro, como se fosse rímel (sem esqcr de selecioná-lo em Mode>Shadows, para reconhecer que deve escurecer as sombras).
